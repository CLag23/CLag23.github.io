Beginner Setup Guide

This page guides you through installing Python, choosing a code editor, and trying online environments. It is read-only, requires no input, and links to reputable external resources for deeper tutorials.

Install Python

Modern tutorials assume Python 3. Visit the official site to get the latest stable release and instructions for your operating system.

  • Official Python Downloads (python.org)
  • Using Python — Platform Guides

Quick Verification

After installation, open your terminal and check the version:

python --version
python3 --version

Either command should print a version number (for example, Python 3.12.x).

Package Installer (pip)

python -m pip --version
python3 -m pip --version

If a version appears, pip is available. See the official docs if you need help:

  • pip Installation
  • Installing Packages Tutorial

Optional: Isolated Project Environments

Virtual environments help keep project dependencies separate. They are optional for beginners but recommended as you start new projects.

  1. Create a folder and a virtual environment: 
    mkdir my_project
cd my_project
python -m venv .venv
  2. Activate it: 
    # Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
  3. Install a package and list what you have: 
    python -m pip install requests
python -m pip list
  4. Deactivate when done: 
    deactivate

More details: Virtual Environments and Packages (docs.python.org)
