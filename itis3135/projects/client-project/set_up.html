Beginner setup guide

Install Python, pick an editor, or run everything in the browser. Keep it simple and verify each step.

Step 1

Install Python 3

Use the official download for your operating system.

  • Python downloads
  • Platform guides

Step 2

Verify quickly

python --version
python3 --version

You should see a version like Python 3.12.x.

Step 3

Check pip

python -m pip --version
python3 -m pip --version

Use pip to install and list packages.

Optional: virtual environments

Keep project dependencies isolated as soon as you start adding packages.

mkdir my_project
cd my_project
python -m venv .venv

# activate
.venv\Scripts\activate
# or on macOS/Linux
source .venv/bin/activate

python -m pip install requests
python -m pip list
deactivate

Sanity check

print("Hello, Python!")

If you see the message in your terminal or notebook, your setup is ready.
