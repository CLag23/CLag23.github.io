Web Development with Python

Ship APIs, dashboards, and full-stack experiences.

Frameworks like Django, FastAPI, and Flask make it fast to build secure, maintainable services. You get URL routing, templating, ORM layers, and auth out of the box.

  • REST + GraphQL APIs: Power mobile apps and single-page apps with predictable endpoints.
  • Full-stack sites: Render server-side pages with reusable components and form handling.
  • Security built-in: Sessions, password hashing, CSRF protection, and permission checks.
  • Database integration: Query PostgreSQL, MySQL, or SQLite cleanly with ORMs.
  • Real-time features: Add websockets for live updates, chat, and notifications.

Popular frameworks: Django, Flask, FastAPI, Pyramid

Real-world examples: Instagram, Spotify, Pinterest, Dropbox

Artificial Intelligence with Python

Go from prototype models to production AI.

Python's ML ecosystem makes experimentation and deployment straightforward. From simple classifiers to deep learning, you can move quickly with strong tooling.

  • Computer vision: Detect objects, track motion, and analyze images or video.

  • Model deployment: Serve models via APIs, queues, or edge devices with monitoring.

Popular libraries: TensorFlow, PyTorch, scikit-learn, Keras, OpenCV

Real-world examples: Recommendation engines, autopilot systems, voice assistants

Web: Simple API Endpoint

A tiny Flask route that returns JSON.

from flask import Flask, jsonify

app = Flask(__name__)

@app.route("/api/users")
def get_users():
    users = [
        {"id": 1, "name": "Alice"},
        {"id": 2, "name": "Bob"},
    ]
    return jsonify(users)

if __name__ == "__main__":
    app.run(debug=True)

AI: Quick Image Classification

Load a pre-trained MobileNet model to label an image.

from tensorflow.keras.applications import MobileNetV2
from tensorflow.keras.preprocessing import image
import numpy as np

model = MobileNetV2(weights="imagenet")

img = image.load_img("cat.jpg", target_size=(224, 224))
x = image.img_to_array(img)
x = np.expand_dims(x, axis=0)

predictions = model.predict(x)
print(predictions[0][:3])

Data: Fast Data Summary

Use pandas to summarize and visualize sales by category.

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("sales.csv")

print(df.describe())

category_totals = df.groupby("category")["sales"].sum()
category_totals.plot(kind="bar", color="#0ea5b7")
plt.title("Sales by Category")
plt.tight_layout()
plt.show()
