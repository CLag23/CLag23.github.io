What is Python?

Python is a high-level, interpreted programming language designed to be readable and expressive. It was created by Guido van Rossum and first released in 1991. The language emphasizes clarity and simplicity, encouraging code that is easy to understand and maintain.

Design Philosophy

Python follows a “readability first” philosophy: clear names, minimal punctuation, and meaningful whitespace. Indentation is part of the syntax, so the structure of a program is visible at a glance. Many of these ideas are captured in the “Zen of Python,” which values simplicity, explicitness, and practicality.

How Python Runs

Python source code is translated to bytecode and executed by a virtual machine. This avoids a separate compile step typical of fully compiled languages. The result is quick iteration and straightforward debugging.

Typing and Variables

  • Dynamically typed: The type of a variable is determined at runtime.
  • Strongly typed: Operations that mix incompatible types cause errors rather than silent conversions.
  • Optional type hints: You can add annotations to improve clarity and tooling, without changing runtime behavior.

Core Syntax Features

  • Indentation defines blocks instead of braces.
  • Colons introduce blocks (after if, for, def, etc.).
  • Whitespace is significant and encourages consistent formatting.

Functions

def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("Ada"))

Exceptions

try:
    result = 10 / 0
except ZeroDivisionError:
    result = None
finally:
    print("Done.")
print(result)

Modules and Imports

Python code is organized into modules (single files) and packages (directories with modules). Imports let you access definitions from other modules, encouraging reuse and clear structure.

import math
print(math.sqrt(16))

Object-Oriented and Beyond

Python supports multiple paradigms: procedural, object-oriented, and functional. You can define classes and create objects, or write concise functions and pure transformations.

class Counter:
    def __init__(self, start: int = 0):
        self.value = start

    def increment(self) -> None:
        self.value += 1

c = Counter()
c.increment()
print(c.value)

Style and Conventions

  • PEP 8: A style guide that recommends naming patterns, line lengths, and formatting.
  • Docstrings: Triple-quoted strings that document modules, classes, and functions.
  • Readability: Prefer explicit code and meaningful names.

Versions and Governance

  • Modern Python commonly refers to the Python 3 series, which introduced improvements over Python 2.
  • The language evolves through proposals called PEPs (Python Enhancement Proposals).
  • The Python Software Foundation (PSF) is a non-profit that manages intellectual property and supports the community.

Summary

Python is a readable, expressive language with clear syntax, significant indentation, and strong yet dynamic typing. Its module system encourages organized programs, while its conventions and philosophy guide developers toward clean, maintainable code.
